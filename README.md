# netflix-data-analysis-project
This project analyzes Netflix movies and TV shows dataset to uncover trends, patterns, and insights about content distribution, genres, ratings, and growth over time.  The goal is to perform data cleaning, exploratory data analysis (EDA), and visualization, and derive meaningful business insights.
# Objectives
1.Clean and preprocess raw Netflix dataset
2.Perform exploratory data analysis (EDA)
3.Identify trends in content type, genres, and countries
4.Analyze content growth over time
5.Generate insights and recommendations
